

# 💬 Chatify – Real-Time Chat Application

## 📌 Overview

**Chatify** is a full-stack real-time chat application that enables users to communicate seamlessly with secure authentication. It provides a responsive and efficient messaging experience using modern web technologies.

live demo: https://chatify-nu-three.vercel.app/

---

## 🚀 Features

* 🔐 **User Authentication & Authorization** (Spring Security)
* 💬 **Real-Time Messaging** between users
* ⚡ **Fast and Responsive UI** using React
* 🔄 **REST API Integration** for client-server communication
* 🗄️ **Database Management** using JDBC
* 🛡️ **Secure Communication** with backend validation

---

## ⚙️ Getting Started

### 🔄 1. Clone the Repository

```bash
git clone https://github.com/your-username/chatify.git
cd chatify
```

---

### ▶️ 2. Run Backend (Spring Boot)

```bash
cd backend
mvn spring-boot:run
```

---

### 🌐 3. Run Frontend (React)

```bash
cd frontend
npm install
npm start
```

---

## 🔐 Authentication

* Users must **sign up/login** to access chat features
* Authentication handled using **Spring Security**

---

## 📡 API Endpoints (Sample)

* `POST /api/auth/signup` → Register user
* `POST /api/auth/login` → Login user
* `GET /api/messages` → Fetch messages
* `POST /api/messages` → Send message
